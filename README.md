# Laboratorio02_DWA
## Descripción: 
Este proyecto consiste en el desarrollo de una aplicación móvil en Android que permite registrar datos personales como nombre, edad y departamento. Los datos ingresados se muestran en una lista y al seleccionar un nombre se visualiza la información adicional.

## Funcionalidades
- Ingresar nombre
- Ingresar edad
- Ingresar departamento
- Agregar datos a una lista
- Mostrar información al seleccionar un nombre
